Autonomous Response

Three-Action Remediation Vocabulary

Different anomaly signatures trigger different actions. Confidence-gated. Cooldown-protected. Every action fully audited.

01
🔄
RESTART
Triggered by: error_burst, new_template, frequency_spike

Confidence gate: > 0.60
Cooldown: 60 seconds
docker.containers.get(svc).restart(timeout=10)
02
⬆️
SCALE UP
Triggered by: resource_saturation, throughput_anomaly

Confidence gate: > 0.75
Adds 1 replica via Compose
docker compose up --scale svc=N+1 -d
03
🛡️
CPU THROTTLE
Triggered by: latency_spike, multi_signal anomaly

Confidence gate: > 0.70
Limits runaway CPU usage
docker update --cpus 0.5 <container>
CONFIDENCE THRESHOLD SYSTEM
HIGH CONFIDENCE · IMMEDIATE ACTION ≥ 0.85
STANDARD CONFIDENCE · EXECUTE 0.60 – 0.85
LOW CONFIDENCE · BLOCKED (FALSE ALERT) < 0.60
Key Features

Every Criterion. Delivered.

Built to satisfy every requirement — from the PS3 spec to live judge scrutiny.

✨ AI · NLP
DistilBERT Severity Classifier
HuggingFace BERT-large-cased-finetuned-mnli wraps Drain3's flagged log templates. Re-scores anomaly severity with <100ms extra latency — only fires when Drain3 already found something.
gchhablani/bert-large-cased-finetuned-mnli drain3 NLI classification
📈 AI · TIME SERIES
LSTM Degradation Detector
2-layer PyTorch LSTM consumes the rolling Prometheus history buffer per service. Outputs a degradation probability score. Runs on simulated data — visible in every evidence packet.
pytorch LSTM 2-layer temporal anomaly
⚡ REMEDIATION
3-Action Remediation Vocab
Beyond just "restart": scale_up adds a Docker Compose replica; cpu_throttle uses Docker's update --cpus API. Each action maps to a distinct anomaly signature with its own confidence gate.
restart scale_up cpu_throttle
⏱️ SLA · MONITORING
Real-Time 15s SLA Gauge
Live cycle timer is prominently displayed on the dashboard. Turns orange at 10s, flashes red at 15s. Judges watching the demo see the SLA constraint being honored in real time.
orange at 10s flashes at 15s grafana annotation
⚔️ CHAOS ENGINEERING
Live Induced Failure Demo

Live Dependency Graph
